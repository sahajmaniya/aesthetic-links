# Aesthetic Link Library

A small, modern, classy micro‑website with a minimalist Apple‑inspired design: clean typography, soft gradients, glass‑morphism cards, and smooth micro‑animations.

## Run locally (optional)
1. Install Node.js (18+ recommended).
2. Install deps:
   ```bash
   npm install
   ```
3. Start dev server:
   ```bash
   npm run dev
   ```

## Deploy to Vercel (no VS Code needed)
### Option A: Upload to GitHub via browser
1. Download the zip from ChatGPT.
2. On GitHub: **New repository** → create it.
3. In the repo page: **Add file → Upload files** → drag the **unzipped** folder contents (not the zip itself).
4. Commit.

### Option B: Deploy on Vercel
1. Go to Vercel → **New Project**
2. Import your GitHub repo
3. Framework: Next.js (auto-detected)
4. Deploy

## Customize
Edit the link list in `lib/links.ts`.
